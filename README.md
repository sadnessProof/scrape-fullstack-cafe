# Scrape-Fullstack-Cafe

### In this repository we are scraping the data from [Fool Stack Cafe](https://www.fullstack.cafe/) using [**Selenium Python**](https://selenium-python.readthedocs.io/). Which provides interview questions for the following fields:
1. **Full-Stack, Web & Mobile**
2. **Coding & Data Structures**
3. **System Design & Architecture**

### For each topic in the field we have different difficulty levels. There are: Entry, Junior, Mid, Senior, and Expert levels. There also are the challenge questions, if you are a little bit evil interviewer ^^
